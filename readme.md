##This is the console based node app.
***********************************

###To run this app in cmd:
***********************

####for adding note:
    node app.js add --title or -t "title of app" --body or -b "body text"

####for listing notes:
    node app.js list 
    
####for reading note:
    node app.js read --title or -t "title of app" 

####for removing note:
    node app.js remove --title or -t "title of app"
